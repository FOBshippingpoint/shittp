shittp

shittp is a shell script that uses tar and base64 to encode your dotfiles and transfer them to remote machines. With shittp, you can feel right at home inside SSH session or Docker container.

Story

Every time I type v and see "command not found." Suddenly realize this is colleague's machine and my v=vim alias is not here. It breaks my heart.

I discovered kyrat, a tool that brings dotfiles to remote SSH sessions. While it was close to what I needed, it lacks Docker support and is strictly Bash-based. As an POSIX shell lover, I decided to build my own alternative.

Usage

  1. 1. Install

    
curl -fsSL https://raw.githubusercontent.com/FOBshippingpoint/shittp/refs/heads/main/install.sh --output install.sh
chmod +x install.sh
./install.sh

  2. 2. Edit Dotfiles

    
cd ~/.config/shittp 
echo 'aloha() { echo hello; }' >> .profile # Add an aloha function
$EDITOR .vimrc
$EDITOR .tmux.conf

  3. 3. Login to Remote Session

    
shittp john@other.machine
john$ aloha  # output: hello
john$ vim    # alias equal to "vim -u $SHITTP_HOME/.vimrc"
john$ tmux   # alias equal to "tmux -f $SHITTP_HOME/.tmux.conf"

    Or Docker container:

    
docker run -it alpine /bin/sh -c "$(shittp print)"
$ aloha    # output: hello
