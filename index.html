shittp

shittp is a shell script that make use of tar, base64 to encode dotfiles to string, and send to another machine. With shittp, you can have at-home experience within SSH connection or Docker environment.

Place asciinema video

Story

When I type v and get "command not found", suddenly realize that this is colleage's machine and no alias v=vim for me. I feel sad.

Then found a tool kyrat, that can bring the dotfiles to remote SSH session, that is what I'm searching for. However, kyrat does not support Docker environment, and is written in bash. As a person who love POSIX Shell, I decide to build an alternative tool.

Usage

1. Install


curl -fsSL https://raw.githubusercontent.com/FOBshippingpoint/shittp/refs/heads/main/install.sh --output install.sh
chmod +x install.sh
./install.sh

2. Edit Dotfiles


cd ~/.config/shittp 
echo 'aloha() { echo hello; }' >> .bashrc # Add an aloha function
$EDITOR .vimrc
$EDITOR .tmux.conf

3. Login to Remote Session


shittp john@other.machine
john$ aloha  # output: hello
john$ vim    # an alias equal to "vim -u $SHITTP_HOME/.vimrc"
john$ tmux   # an alias equal to "tmux -f $SHITTP_HOME/.tmux.conf"
